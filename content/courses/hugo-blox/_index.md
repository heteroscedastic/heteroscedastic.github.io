Welcome to the Hugo Blox introductory course! This course will teach you the basics of creating a website with Hugo Blox.

## Getting Started

Hugo Blox is a website builder for Hugo that enables you to create any kind of academic or personal website without needing to code.

### Key Features

- **No coding required** - Use the visual editor in VS Code
- **Markdown content** - Write content in simple Markdown
- **Beautiful themes** - Choose from multiple color themes
- **Fast & SEO-optimized** - 100/100 Lighthouse scores
