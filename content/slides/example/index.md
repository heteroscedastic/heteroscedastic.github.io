## Features

- Write slides in Markdown
- Dark / Light mode toggle
- Speaker notes
- LaTeX math and code highlighting

---

## Controls

- Next: `Right Arrow` or `Space`
- Previous: `Left Arrow`
- Start: `Home`
- Finish: `End`
- Overview: `Esc`
- Speaker notes: `S`
- Fullscreen: `F`
- Zoom: `Alt + Click`
